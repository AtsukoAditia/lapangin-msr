# 01 — Project Overview

## Nama Project

Lapangin

## Jenis Aplikasi

Aplikasi booking lapangan olahraga berbasis web PWA dengan admin CMS.

## Olahraga yang Didukung

- Futsal
- Minisoccer
- Badminton
- Padel
- Tenis
- Basket
- Voli
- Olahraga lain via master data

## Core Value

Customer:

- Cepat lihat jadwal kosong.
- Cepat booking.
- Dapat bukti booking.

Admin/Owner:

- Kelola jadwal.
- Hindari bentrok booking.
- Pantau pendapatan.
- Kelola lapangan dan harga.

## Modul Utama

1. Public Website
2. Booking Engine
3. Admin CMS
4. Pricing Engine
5. Availability Engine
6. Notification
7. PWA
8. Database Adapter
9. Reports
10. Migration to PostgreSQL
