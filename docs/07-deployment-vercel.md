# 07 — Deployment Vercel

## Langkah

1. Push project ke GitHub.
2. Login ke Vercel.
3. Import repository.
4. Pilih framework Next.js.
5. Tambahkan environment variables.
6. Deploy.
7. Test URL preview.
8. Promote ke production.

## Environment Variables

Isi di Vercel Project Settings:

- NEXT_PUBLIC_APP_NAME
- NEXT_PUBLIC_APP_URL
- DATABASE_PROVIDER
- GOOGLE_SHEETS_SPREADSHEET_ID
- GOOGLE_SHEETS_CLIENT_EMAIL
- GOOGLE_SHEETS_PRIVATE_KEY

## Checklist Setelah Deploy

- Homepage bisa dibuka.
- Booking page bisa dibuka.
- Admin page bisa dibuka.
- Submit booking berhasil.
- Data masuk Spreadsheet.
- Tidak ada error serverless.
- Mobile responsive.
